# 场景拆分 (Split View)

本示例展示了如何在一个 Canvas 中实现**双目视差**或**分屏监控**的效果。

## 实现思路

1.  **多相机渲染**：
    *   创建两个 `PerspectiveCamera`（左眼和右眼）。
    *   分别设置左右偏移量（`position.x = ±0.3`），模拟人眼的瞳距。

2.  **视口裁剪 (Scissor Test)**：
    *   使用 WebGL 的 `gl.setScissor` 和 `gl.setViewport` 方法，将 Canvas 划分为左右两个独立的渲染区域。
    *   在每一帧渲染循环中，先清空屏幕（手动管理 `autoClear`），然后分别激活左/右视口，用对应的相机渲染同一个场景。

3.  **性能优化**：
    *   所有渲染逻辑在 `useFrame` 中完成，确保高效的逐帧更新。
    *   因为是同一个 `scene` 被渲染两次，所以不需要创建额外的场景对象，内存开销极低。
