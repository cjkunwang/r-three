# 材质对比 (Material Split)

本示例展示了常见的“推拉式”对比效果，常用于展示画质优化前后、材质差异或不同渲染风格的对比。

## 实现思路

1.  **无缝拼接**：
    *   为了实现“同一个物体换皮”的错觉，左右两个相机必须拥有完全一致的**透视关系**。
    *   关键点：`gl.setViewport` 始终设置为全屏大小（而不是像前两个例子那样减半）。这样计算出的投影矩阵是相同的。
    *   通过 `gl.setScissor` 仅控制“可见区域”，像遮罩一样裁掉多余部分。

2.  **动态分割线**：
    *   实现了一个支持鼠标/触摸拖拽的 `Slider` 组件，计算出 0~1 之间的分割比例 `split`。
    *   在渲染循环中，根据 `split` 动态计算裁剪矩形的宽度：
        *   左屏：`width * split`
        *   右屏：`width * (1 - split)`，起始位置为 `width * split`

3.  **场景复用与差异化**：
    *   同样使用了 `createPortal` 和双场景架构。
    *   左右场景加载了完全相同的几何体（Geometry）和动画逻辑。
    *   但在右侧场景中，应用了更高级的 `MeshPhysicalMaterial` 和额外的环境光照，以突出材质差异。
